emolRTE - Even MOre Lightweight Rich Text Editor

Unobtrusive, unbloated Rich Text Editor for JQuery. It should work with all major browsers (IE 6+, Firefox 2+, Opera 9+, Safari 3+) and degrades gracefully to textarea field for others.

Needs more testing. Feedback very welcome.

Various browsers may output different HTML code.
